# unidiff-rs

Unified diff parsing/metadata extraction library for Rust

## Installation

Add it to your ``Cargo.toml``:

```toml
[dependencies]
unidiff = "0.3"
```

Add ``extern crate unidiff`` to your crate root and your're good to go!
